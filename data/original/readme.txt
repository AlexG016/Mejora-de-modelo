Data can be manually downloaded from here:
- MIT_BIH_Arrhythmia : https://doi.org/10.13026/C2F305
- MIT_BIH_LongTerm : https://doi.org/10.13026/C2KS3F
- MIT_BIH_NormalSinus : https://doi.org/10.13026/C2NK5R
- WCS_dataset : https://doi.org/10.21979/N9/42BBFA

or automatically downloaded with the download_MIT-BIH_datasets.py script.

They should be separately extracted in this folder in the respective sub-folders.
